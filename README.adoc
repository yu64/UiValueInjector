# UiValueInjector

## 概要

UI自動化を用いて、UI要素に値を設定するアプリ。

「このGUIアプリ便利なんだけど毎回ファイルパスを入力するのが面倒」
そんな悩みを解決します。


## ルール設定ファイルの例

[source, yaml]
----
rules:
  - name: RuleName
    timing: Always
    elements:
      - type: automationId
        value: txtBoxId
    value: "in_{0}"
----


## 不足機能

. すでに存在するプロセスに対する値の設定
. 起動するプロセスにコマンドライン引数を渡す
